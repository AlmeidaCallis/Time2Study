# 📚 Time2Study

A sleek, modern React application designed to help students track and analyze their study habits with a focus on productivity and aesthetics.

## ✨ Features

- **Dynamic Tracking**: Log subject, duration, and date with ease.
- **Categorization**: Tag sessions as *Reading*, *Assignment*, *Revision*, or *Exam Prep*.
- **Real-time Filtering**: Search through your study history instantly.
- **Glassmorphism UI**: High-quality design featuring a study-focused background.
- **Dark Mode**: Optimized color palette for late-night sessions.
- **Auto-Calculations**: Dynamic summary of total hours and subject-wise breakdown.

## 🚀 Tech Stack

- **Frontend**: React.js (Functional Components & Hooks)
- **Styling**: CSS3 (Custom Variables & Backdrop Filters)
- **State Management**: React `useState` & `reduce()` for real-time analytics.
